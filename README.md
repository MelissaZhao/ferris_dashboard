# Ferris Dashboard Project1

## Requirements
- Python 3.x
- Kafka: Download and install from [Kafka official website](https://kafka.apache.org/downloads)

## Kafka Setup
1. Download Kafka from the [official website](https://kafka.apache.org/downloads) and extract the contents.
2. Follow the [official documentation](https://kafka.apache.org/quickstart) to set up and start the Kafka Zookeeper and Kafka server.
3. Start the Kafka producer and consumer as per your project requirements.



## Setup
1. Clone this repository.
2. Install Python dependencies: `pip install -r requirements.txt`
3. Start the Kafka Zookeeper and Kafka server.
4. Start the Kafka producer and consumer.
5. Give the `run.sh` script executable permissions: `chmod +x run.sh`
6. Run `run.sh` to start the `scan_news_keywords.py` and `app.py` scripts.

## Configuration
Edit the `config.json` file to change the keywords for `scan_news_keywords.py`.
